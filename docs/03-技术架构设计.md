# 技术架构设计

## 1. 技术栈选型

### 1.1 核心技术栈

| 技术 | 选型 | 说明 |
|------|------|------|
| 运行时 | .NET 10 | 最新 LTS 版本，性能优秀 |
| 编程语言 | C# | 强类型，生态丰富 |
| LLM 接口 | OpenAI API 兼容 | 灵活切换不同供应商 |
| AI 封装 | Microsoft.Extensions.AI | 轻量级，不引入重框架 |
| 向量存储 | AWS Knowledge Base | 托管服务，运维成本低 |
| 数据库 | PostgreSQL | 可靠稳定，支持 JSON |
| ORM | EF Core | .NET 生态标准 ORM |

### 1.2 技术选型原则

**为什么不用 Microsoft Agent Framework？**
- 框架过重，引入不必要的复杂性
- 学习曲线陡峭
- 定制化困难
- Microsoft.Extensions.AI 足够满足需求

**为什么选择 AWS Knowledge Base？**
- 托管服务，无需自建向量数据库
- 与 AWS 生态集成良好
- 支持多种数据源接入
- 内置 RAG 能力

### 2.2 架构分层

| 层次 | 职责 | 说明 |
|------|------|------|
| Application Layer | 具体业务实现 | SRE 相关的 Agent 和 Tool 实现 |
| Agent Framework | 通用 Agent 框架 | 提供抽象、基础实现、Hooks 等 |
| Infrastructure | 基础设施 | 数据库、LLM、向量存储等 |

### 2.3 关键设计原则

1. **框架与业务分离**：Agent Framework 不包含任何具体业务逻辑
2. **可扩展性**：通过接口和 Hooks 允许业务层自定义行为
3. **全局 Result 模式**：所有操作返回 Result，不抛异常，保证错误信息完整传递
4. **轻量级通信**：使用 SSE 而非 SignalR 等重框架

## 3. 全局 Result 模式

### 3.1 设计理念

Result 模式不仅应用于 Tool 调用，而是**整个系统的统一错误处理策略**：

| 场景 | Result 类型 | 说明 |
|------|-------------|------|
| Agent 执行 | `AgentResult` | Agent 整体执行结果 |
| LLM 调用 | `LLMResult` | 大模型 API 调用结果 |
| Tool 执行 | `ToolResult` | 工具执行结果 |
| 知识库检索 | `Result<T>` | 通用结果包装 |

**为什么全局使用 Result 模式？**

1. **错误信息不丢失**：异常会中断流程，而 Result 保留完整上下文
2. **LLM 可感知错误**：LLM 能根据错误信息自行调整策略
3. **统一处理风格**：整个系统一致的错误处理方式
4. **支持重试决策**：`IsRetryable` 让调用方知道是否值得重试
5. **便于日志和监控**：结构化的错误信息更易追踪

### 3.2 通用 Result 基类

```csharp
/// <summary>
/// 通用 Result 基类
/// </summary>
public abstract class ResultBase
{
    /// <summary>是否成功</summary>
    public bool IsSuccess { get; init; }
    
    /// <summary>错误代码</summary>
    public string? ErrorCode { get; init; }
    
    /// <summary>错误消息</summary>
    public string? ErrorMessage { get; init; }
    
    /// <summary>是否可重试</summary>
    public bool IsRetryable { get; init; }
    
    /// <summary>执行耗时</summary>
    public TimeSpan Duration { get; init; }
    
    /// <summary>元数据</summary>
    public Dictionary<string, object> Metadata { get; init; } = new();
}

/// <summary>
/// 带数据的通用 Result
/// </summary>
public class Result<T> : ResultBase
{
    /// <summary>结果数据</summary>
    public T? Data { get; init; }
    
    public static Result<T> Success(T data, TimeSpan? duration = null)
        => new()
        {
            IsSuccess = true,
            Data = data,
            Duration = duration ?? TimeSpan.Zero
        };
    
    public static Result<T> Failure(
        string errorMessage,
        string? errorCode = null,
        bool isRetryable = true)
        => new()
        {
            IsSuccess = false,
            ErrorMessage = errorMessage,
            ErrorCode = errorCode,
            IsRetryable = isRetryable
        };
    
    public static Result<T> FromException(Exception ex, bool? isRetryable = null)
        => new()
        {
            IsSuccess = false,
            ErrorMessage = ex.Message,
            ErrorCode = ex.GetType().Name,
            IsRetryable = isRetryable ?? IsExceptionRetryable(ex),
            Metadata = new Dictionary<string, object>
            {
                ["exception_type"] = ex.GetType().FullName ?? "",
                ["stack_trace"] = ex.StackTrace ?? ""
            }
        };
    
    private static bool IsExceptionRetryable(Exception ex) => ex switch
    {
        TimeoutException => true,
        HttpRequestException => true,
        OperationCanceledException => false,
        ArgumentException => false,
        _ => true
    };
}
```

### 3.3 LLM 调用 Result

```csharp
/// <summary>
/// LLM 调用结果
/// </summary>
public class LLMResult : ResultBase
{
    /// <summary>响应消息</summary>
    public Message? Message { get; init; }
    
    /// <summary>Token 使用量</summary>
    public TokenUsage TokenUsage { get; init; } = new();
    
    /// <summary>使用的模型</summary>
    public string? Model { get; init; }
    
    /// <summary>完成原因</summary>
    public string? FinishReason { get; init; }
    
    public static LLMResult Success(
        Message message,
        TokenUsage tokenUsage,
        string model,
        string? finishReason = null,
        TimeSpan? duration = null)
        => new()
        {
            IsSuccess = true,
            Message = message,
            TokenUsage = tokenUsage,
            Model = model,
            FinishReason = finishReason,
            Duration = duration ?? TimeSpan.Zero
        };
    
    public static LLMResult Failure(
        string errorMessage,
        string errorCode,
        bool isRetryable = true)
        => new()
        {
            IsSuccess = false,
            ErrorMessage = errorMessage,
            ErrorCode = errorCode,
            IsRetryable = isRetryable
        };
    
    /// <summary>从 API 异常创建失败结果</summary>
    public static LLMResult FromException(Exception ex)
    {
        var (code, retryable) = ClassifyException(ex);
        return new LLMResult
        {
            IsSuccess = false,
            ErrorMessage = ex.Message,
            ErrorCode = code,
            IsRetryable = retryable,
            Metadata = new Dictionary<string, object>
            {
                ["exception_type"] = ex.GetType().FullName ?? ""
            }
        };
    }
    
    private static (string Code, bool Retryable) ClassifyException(Exception ex)
    {
        return ex switch
        {
            HttpRequestException http when http.Message.Contains("429") 
                => ("RATE_LIMIT", true),
            HttpRequestException http when http.Message.Contains("503") 
                => ("SERVICE_UNAVAILABLE", true),
            HttpRequestException http when http.Message.Contains("401") 
                => ("UNAUTHORIZED", false),
            HttpRequestException http when http.Message.Contains("400") 
                => ("BAD_REQUEST", false),
            TimeoutException 
                => ("TIMEOUT", true),
            TaskCanceledException 
                => ("CANCELLED", false),
            JsonException 
                => ("INVALID_RESPONSE", true),
            _ => ("UNKNOWN", true)
        };
    }
}

public record TokenUsage(int PromptTokens = 0, int CompletionTokens = 0)
{
    public int TotalTokens => PromptTokens + CompletionTokens;
    
    public static TokenUsage operator +(TokenUsage a, TokenUsage b)
        => new(a.PromptTokens + b.PromptTokens, a.CompletionTokens + b.CompletionTokens);
}
```

### 3.4 LLM 客户端包装

```csharp
/// <summary>
/// LLM 客户端接口（返回 Result）
/// </summary>
public interface ILLMClient
{
    /// <summary>执行 Chat Completion</summary>
    Task<LLMResult> CompleteAsync(
        IReadOnlyList<Message> messages,
        LLMOptions options,
        CancellationToken cancellationToken = default);
}

public class LLMOptions
{
    public string Model { get; set; } = "gpt-4";
    public double Temperature { get; set; } = 0.7;
    public int? MaxTokens { get; set; }
    public IReadOnlyList<ITool>? Tools { get; set; }
}

/// <summary>
/// LLM 客户端实现（包装 Microsoft.Extensions.AI）
/// </summary>
public class LLMClient : ILLMClient
{
    private readonly IChatClient _chatClient;
    private readonly ILogger<LLMClient> _logger;
    
    public async Task<LLMResult> CompleteAsync(
        IReadOnlyList<Message> messages,
        LLMOptions options,
        CancellationToken cancellationToken = default)
    {
        var sw = Stopwatch.StartNew();
        
        try
        {
            var chatMessages = ConvertMessages(messages);
            var chatOptions = BuildChatOptions(options);
            
            var response = await _chatClient.CompleteAsync(
                chatMessages, 
                chatOptions, 
                cancellationToken);
            
            sw.Stop();
            
            var resultMessage = ConvertResponse(response);
            var tokenUsage = new TokenUsage(
                response.Usage?.InputTokenCount ?? 0,
                response.Usage?.OutputTokenCount ?? 0);
            
            return LLMResult.Success(
                resultMessage,
                tokenUsage,
                options.Model,
                response.FinishReason?.ToString(),
                sw.Elapsed);
        }
        catch (OperationCanceledException)
        {
            return LLMResult.Failure(
                "LLM request was cancelled",
                "CANCELLED",
                isRetryable: false);
        }
        catch (Exception ex)
        {
            sw.Stop();
            _logger.LogError(ex, "LLM call failed after {Duration}ms", sw.ElapsedMilliseconds);
            
            return LLMResult.FromException(ex) with { Duration = sw.Elapsed };
        }
    }
}
```

### 3.7 Result 在 Agent 执行中的应用

```csharp
public class ToolLoopAgent : IAgent
{
    public async Task<AgentResult> ExecuteAsync(
        AgentExecutionContext context,
        CancellationToken cancellationToken = default)
    {
        var allMessages = new List<Message>();
        var totalTokenUsage = new TokenUsage();
        
        for (int iteration = 0; iteration < _options.MaxIterations; iteration++)
        {
            // 调用 LLM（返回 Result，不抛异常）
            var llmResult = await _llmClient.CompleteAsync(
                messages, options, cancellationToken);
            
            // 处理 LLM 调用失败
            if (!llmResult.IsSuccess)
            {
                // 如果可重试，尝试重试
                if (llmResult.IsRetryable && iteration < _options.MaxIterations - 1)
                {
                    _logger.LogWarning(
                        "LLM call failed with {ErrorCode}, retrying...", 
                        llmResult.ErrorCode);
                    await Task.Delay(GetRetryDelay(llmResult), cancellationToken);
                    continue;
                }
                
                // 不可重试或达到最大重试次数，返回失败
                return AgentResult.FromLLMError(llmResult, allMessages);
            }
            
            totalTokenUsage += llmResult.TokenUsage;
            allMessages.Add(llmResult.Message!);
            
            // 处理工具调用
            var toolCalls = llmResult.Message!.Parts.OfType<ToolCallPart>().ToList();
            if (toolCalls.Any())
            {
                var toolResults = await ExecuteToolCallsAsync(toolCalls, cancellationToken);
                
                // 工具结果（无论成功失败）都添加到消息中供 LLM 参考
                var toolResultMessage = CreateToolResultMessage(toolResults);
                allMessages.Add(toolResultMessage);
            }
            else
            {
                // 没有工具调用，Agent 完成
                var output = llmResult.Message.Parts
                    .OfType<TextPart>()
                    .FirstOrDefault()?.Text;
                
                return AgentResult.Success(
                    output ?? "",
                    allMessages,
                    totalTokenUsage,
                    iteration + 1);
            }
        }
        
        return AgentResult.Failure(
            "Reached maximum iterations without completion",
            "MAX_ITERATIONS",
            allMessages);
    }
    
    private TimeSpan GetRetryDelay(LLMResult result)
    {
        return result.ErrorCode switch
        {
            "RATE_LIMIT" => TimeSpan.FromSeconds(30),
            "SERVICE_UNAVAILABLE" => TimeSpan.FromSeconds(5),
            "TIMEOUT" => TimeSpan.FromSeconds(2),
            _ => TimeSpan.FromSeconds(1)
        };
    }
}
```

## 4. Agent Framework 设计

### 4.1 核心抽象

#### 4.1.1 Agent 接口

```csharp
/// <summary>
/// Agent 的核心抽象接口
/// Agent 是无状态的执行引擎，对话状态由 IContextManager 管理
/// </summary>
```

```csharp
/// <summary>
/// 强类型工具基类
/// 自动处理 JSON 序列化和参数验证
/// </summary>
public abstract class ToolBase<TParams> : ITool where TParams : class, new()
{
    public abstract string Name { get; }
    public abstract string Summary { get; }
    public abstract string Description { get; }
    public virtual string Category => "General";
    protected virtual string? ReturnDescription => null;
    
    /// <summary>获取工具详情（自动生成 JSON Schema）</summary>
    public ToolDetail GetDetail()
    {
        var schema = SchemaGenerator.GenerateSchema(typeof(TParams));
        return new ToolDetail(Name, Description, schema, ReturnDescription);
    }
    
    /// <summary>执行工具（处理反序列化、验证、执行）</summary>
    public async Task<ToolResult> ExecuteAsync(
        ToolExecutionContext context,
        CancellationToken cancellationToken = default)
    {
        // 1. 反序列化参数
        var (success, parameters, error) = DeserializeParameters(context.Parameters);
        if (!success)
        {
            return ToolResult.Failure(error!, "INVALID_PARAMETERS", isRetryable: true);
        }
        
        // 2. 验证参数（使用 DataAnnotations）
        var validationResults = ValidateParameters(parameters!);
        if (validationResults.Any())
        {
            var errorMessage = string.Join("; ", validationResults.Select(v => v.ErrorMessage));
            return ToolResult.Failure(errorMessage, "VALIDATION_ERROR", isRetryable: true);
        }
        
        // 3. 执行核心逻辑
        return await ExecuteAsync(parameters!, context, cancellationToken);
    }
    
    /// <summary>子类实现的核心逻辑</summary>
    protected abstract Task<ToolResult> ExecuteAsync(
        TParams parameters,
        ToolExecutionContext context,
        CancellationToken cancellationToken);
}
```

#### 3.5.1 ToolLoopAgent（核心基础实现）

```csharp
/// <summary>
/// 工具循环 Agent - Framework 提供的基础实现
/// 实现标准的 ReAct 模式：Think -> Act -> Observe 循环
/// </summary>
public class ToolLoopAgent : IAgent
{
    private readonly IChatClient _chatClient;
    private readonly AgentOptions _options;
    private readonly IAgentHooks _hooks;
    private readonly IContextManager _contextManager;
    private readonly IEventBus _eventBus;
    
    public string Id { get; }
    public string Name { get; }
    public string Description { get; }
    
    public ToolLoopAgent(
        string id,
        string name,
        string description,
        IChatClient chatClient,
        AgentOptions options,
        IAgentHooks? hooks = null,
        IEventBus? eventBus = null)
    {
        Id = id;
        Name = name;
        Description = description;
        _chatClient = chatClient;
        _options = options;
        _hooks = hooks ?? new DefaultAgentHooks();
        _contextManager = options.ContextManager ?? new DefaultContextManager(new TiktokenEstimator());
        _eventBus = eventBus ?? new NullEventBus();
    }
    
    public async Task<AgentResult> ExecuteAsync(
        AgentExecutionContext context,
        CancellationToken cancellationToken = default)
    {
        var allMessages = new List<Message>();
        var totalTokenUsage = new TokenUsage();
        
        try
        {
            // Before Execute Hook
            var beforeCtx = new BeforeExecuteContext
            {
                SessionId = context.SessionId,
                AgentId = Id,
                Input = context.Input,
                Options = _options
            };
            var hookResult = await _hooks.OnBeforeExecuteAsync(beforeCtx);
            if (!hookResult.ShouldContinue)
            {
                return AgentResult.Failure(
                    new AgentError("ABORTED", hookResult.AbortReason ?? "Aborted by hook"),
                    allMessages);
            }
            
            // 初始化上下文
            await InitializeContextAsync(context, cancellationToken);
            
            // 主循环
            for (int iteration = 0; iteration < _options.MaxIterations; iteration++)
            {
                cancellationToken.ThrowIfCancellationRequested();
                
                // 获取用于 LLM 的消息
                var messagesForLLM = await _contextManager.GetMessagesForCompletionAsync(
                    _options.MaxTokens ?? 100000, cancellationToken);
                
                // Before LLM Call Hook
                var llmContext = new BeforeLLMCallContext
                {
                    SessionId = context.SessionId,
                    AgentId = Id,
                    IterationNumber = iteration,
                    Model = _options.Model,
                    Temperature = _options.Temperature,
                    MaxTokens = _options.MaxTokens,
                    Messages = messagesForLLM.ToList(),
                    Tools = _options.Tools.ToList()
                };
                
                hookResult = await _hooks.OnBeforeLLMCallAsync(llmContext);
                if (!hookResult.ShouldContinue)
                {
                    return AgentResult.Failure(
                        new AgentError("ABORTED", hookResult.AbortReason ?? "Aborted by hook"),
                        allMessages);
                }
                
                // 调用 LLM
                var llmResponse = await CallLLMAsync(llmContext, cancellationToken);
                totalTokenUsage = totalTokenUsage with
                {
                    PromptTokens = totalTokenUsage.PromptTokens + llmResponse.TokenUsage.PromptTokens,
                    CompletionTokens = totalTokenUsage.CompletionTokens + llmResponse.TokenUsage.CompletionTokens
                };
                
                // 添加响应消息
                await _contextManager.AddMessageAsync(llmResponse.Message, cancellationToken);
                allMessages.Add(llmResponse.Message);
                
                // After LLM Call Hook
                await _hooks.OnAfterLLMCallAsync(new AfterLLMCallContext
                {
                    SessionId = context.SessionId,
                    AgentId = Id,
                    IterationNumber = iteration,
                    Model = llmContext.Model,
                    Response = llmResponse.Message,
                    TokenUsage = llmResponse.TokenUsage,
                    Duration = llmResponse.Duration
                });
                
                // 检查是否有工具调用
                if (llmResponse.Message.Parts.OfType<ToolCallPart>().Any())
                {
                    // 执行工具调用
                    var toolResults = await ExecuteToolCallsAsync(
                        context.SessionId, 
                        llmResponse.Message.Parts.OfType<ToolCallPart>().ToList(),
                        context.Variables,
                        cancellationToken);
                    
                    // 添加工具结果消息
                    var toolResultMessage = CreateToolResultMessage(toolResults);
                    await _contextManager.AddMessageAsync(toolResultMessage, cancellationToken);
                    allMessages.Add(toolResultMessage);
                }
                else
                {
                    // 没有工具调用，检查是否完成
                    var textContent = llmResponse.Message.Parts
                        .OfType<TextPart>()
                        .Select(p => p.Text)
                        .FirstOrDefault();
                    
                    if (!string.IsNullOrEmpty(textContent))
                    {
                        // Agent 完成
                        await _hooks.OnAfterExecuteAsync(new AfterExecuteContext
                        {
                            SessionId = context.SessionId,
                            AgentId = Id,
                            Result = AgentResult.Success(textContent, allMessages),
                            TotalDuration = TimeSpan.Zero // TODO: track duration
                        });
                        
                        return AgentResult.Success(textContent, allMessages) with
                        {
                            TokenUsage = totalTokenUsage
                        };
                    }
                }
                
                // Iteration End Hook
                var iterEndCtx = new IterationEndContext
                {
                    SessionId = context.SessionId,
                    AgentId = Id,
                    IterationNumber = iteration,
                    MaxIterations = _options.MaxIterations,
                    TotalTokenUsage = totalTokenUsage,
                    ShouldContinue = true
                };
                
                hookResult = await _hooks.OnIterationEndAsync(iterEndCtx);
                if (!hookResult.ShouldContinue || !iterEndCtx.ShouldContinue)
                {
                    break;
                }
            }
            
            // 达到最大迭代次数
            return AgentResult.Failure(
                new AgentError("MAX_ITERATIONS", "Reached maximum iterations without completion"),
                allMessages) with { TokenUsage = totalTokenUsage };
        }
        catch (Exception ex)
        {
            await _hooks.OnErrorAsync(new ErrorContext
            {
                SessionId = context.SessionId,
                AgentId = Id,
                Exception = ex,
                Phase = "execution"
            });
            
            return AgentResult.Failure(
                new AgentError("EXCEPTION", ex.Message, ex),
                allMessages) with { TokenUsage = totalTokenUsage };
        }
    }
    
    private async Task<List<(string ToolCallId, ToolResult Result)>> ExecuteToolCallsAsync(
        Guid sessionId,
        List<ToolCallPart> toolCalls,
        IReadOnlyDictionary<string, object> variables,
        CancellationToken cancellationToken)
    {
        var results = new List<(string, ToolResult)>();
        
        foreach (var toolCall in toolCalls)
        {
            var tool = _options.Tools.FirstOrDefault(t => t.Name == toolCall.Name);
            
            ToolResult result;
            if (tool == null)
            {
                // 工具不存在，返回错误结果（不抛异常）
                result = ToolResult.Failure(
                    $"Tool '{toolCall.Name}' not found. Available tools: {string.Join(", ", _options.Tools.Select(t => t.Name))}",
                    "TOOL_NOT_FOUND");
            }
            else
            {
                // Before Tool Call Hook
                var beforeCtx = new BeforeToolCallContext
                {
                    SessionId = sessionId,
                    AgentId = Id,
                    ToolName = toolCall.Name,
                    Parameters = JsonDocument.Parse(toolCall.Arguments).RootElement
                };
                
                var hookResult = await _hooks.OnBeforeToolCallAsync(beforeCtx);
                if (!hookResult.ShouldContinue)
                {
                    result = ToolResult.Failure(
                        hookResult.AbortReason ?? "Tool call aborted by hook",
                        "ABORTED_BY_HOOK",
                        isRetryable: false);
                }
                else
                {
                    // 执行工具（捕获所有异常，转为 Result）
                    var sw = System.Diagnostics.Stopwatch.StartNew();
                    try
                    {
                        var toolContext = new ToolExecutionContext
                        {
                            SessionId = sessionId,
                            AgentId = Id,
                            Parameters = beforeCtx.Parameters,
                            RawArguments = toolCall.Arguments,
                            Variables = variables
                        };
                        
                        result = await tool.ExecuteAsync(toolContext, cancellationToken);
                    }
                    catch (Exception ex)
                    {
                        // 即使工具抛异常，也转为 Result 返回给 LLM
                        result = ToolResult.FromException(ex);
                    }
                    sw.Stop();
                    result = result with { Duration = sw.Elapsed };
                    
                    // After Tool Call Hook
                    await _hooks.OnAfterToolCallAsync(new AfterToolCallContext
                    {
                        SessionId = sessionId,
                        AgentId = Id,
                        ToolName = toolCall.Name,
                        Result = result,
                        Duration = sw.Elapsed
                    });
                }
            }
            
            results.Add((toolCall.Id, result));
        }
        
        return results;
    }
    
    // ... 其他私有方法
}
```

```csharp
/// <summary>
/// 将 Agent 包装为 Tool，用于多 Agent 协作

public class AgentAsTool : ITool

    private readonly IAgent _agent;
    private readonly AgentAsToolOptions _options;

    public string Description { get; }
    public JsonElement ParameterSchema { get; }
    
    public AgentAsTool(IAgent agent, AgentAsToolOptions? options = null)
    {
        _agent = agent;
        _options = options ?? new AgentAsToolOptions();
        
        Name = options?.ToolName ?? $"delegate_to_{agent.Id}";
        Description = options?.ToolDescription ?? agent.Description;
        ParameterSchema = BuildParameterSchema();
    }
    
    public async Task<ToolResult> ExecuteAsync(
        ToolExecutionContext context,
        CancellationToken cancellationToken = default)
    {
        try
        {
            var input = context.Parameters.GetProperty("task").GetString() ?? "";
            
            var agentContext = new AgentExecutionContext
            {
                SessionId = context.SessionId,
                Input = input,
                Variables = new Dictionary<string, object>(context.Variables),
                ParentAgentId = context.AgentId
            };
            
            var result = await _agent.ExecuteAsync(agentContext, cancellationToken);
            
            if (result.IsSuccess)
            {
                return ToolResult.Success(
                    result.Output ?? "Agent completed without output",
                    new { AgentId = _agent.Id, Messages = result.Messages });
            }
            else
            {
                return ToolResult.Failure(
                    $"Agent failed: {result.Error?.Message}",
                    result.Error?.Code,
                    isRetryable: true);
            }
        }
        catch (Exception ex)
        {
            return ToolResult.FromException(ex);
        }
    }

    {
        var schema = new
        {
            type = "object",
            properties = new
            {
                task = new
                {
                    type = "string",
                    description = $"The task to delegate to {_agent.Name}"
                }
            },
            required = new[] { "task" }
        };
        
        return JsonSerializer.SerializeToElement(schema);
```

```csharp
/// <summary>
/// SSE 事件流写入器
/// </summary>
public interface ISseWriter
{
    Task WriteEventAsync<T>(T @event, CancellationToken cancellationToken = default)
        where T : IAgentEvent;
    
    Task WriteDataAsync(string data, string? eventType = null, 
        CancellationToken cancellationToken = default);
    
    Task FlushAsync(CancellationToken cancellationToken = default);
}

/// <summary>
/// SSE 事件流实现
/// </summary>
public class SseWriter : ISseWriter
{
    private readonly Stream _stream;
    private readonly StreamWriter _writer;
    
    public SseWriter(Stream stream)
    {
        _stream = stream;
        _writer = new StreamWriter(stream, Encoding.UTF8, leaveOpen: true);
    }
    
    public async Task WriteEventAsync<T>(T @event, CancellationToken cancellationToken)
        where T : IAgentEvent
    {
        var eventType = typeof(T).Name.ToLowerInvariant();
        var data = JsonSerializer.Serialize(@event);
        await WriteDataAsync(data, eventType, cancellationToken);
    }
    
    public async Task WriteDataAsync(string data, string? eventType, 
        CancellationToken cancellationToken)
    {
        if (!string.IsNullOrEmpty(eventType))
        {
            await _writer.WriteLineAsync($"event: {eventType}");
        }
        
        // SSE 数据行
        foreach (var line in data.Split('\n'))
        {
            await _writer.WriteLineAsync($"data: {line}");
        }
        
        await _writer.WriteLineAsync(); // 空行表示事件结束
        await _writer.FlushAsync(cancellationToken);
    }
    
    public async Task FlushAsync(CancellationToken cancellationToken)
    {
        await _writer.FlushAsync(cancellationToken);
    }
}
```

```csharp
                await sseWriter.WriteEventAsync(@event, ct);

        });
        
        // 执行 Agent

        // 发送完成事件

            JsonSerializer.Serialize(result),
```

## 8. 总结

### 8.1 架构要点

1. **Framework 与业务分离**
   - Agent Framework 只提供抽象和基础实现
   - 不包含任何 SRE 具体业务逻辑
   - 业务 Agent 在 Application Layer 实现

2. **Result 模式**
   - 所有 Tool 执行返回 `ToolResult`，不抛异常
   - 确保 LLM 能获得完整的错误信息进行重试

3. **Hooks 系统**
   - 允许业务层在 Agent 执行各阶段介入
   - 支持动态修改模型、参数等
   - 默认实现为空操作，易于扩展

4. **三层数据模型**
   - Session → Message → Part
   - 支持多模态内容
   - 灵活的元数据扩展

5. **轻量级通信**
   - 使用 SSE 替代 SignalR
   - Framework 层不引入重量级框架
   - 业务层可根据需要选择通信方式
