# 上下文管理策略

## 1. 概述

上下文管理是 AI Agent 系统的核心挑战之一。Agent Framework 提供：
- **充血模型**：`IContextManager` 采用充血模型，封装完整的上下文管理逻辑
- **默认实现**：`DefaultContextManager` 满足通用需求
- **Token 限制管理**：基于模型最大 Token 数的自动剪枝
- **可插拔剪枝策略**：`IContextTrimmer` 接口支持自定义剪枝逻辑
- **上下文快照**：`ContextSnapshot` 支持断点恢复和多 Agent 协作
- **工厂方法**：支持多种创建方式（`StartNew`、`FromSnapshot`、`Continue`）

## 2. 上下文来源类型系统 (ContextSource)

### 2.1 设计动机

在 Agent 执行时，上下文可能来自多种来源：
- 全新对话
- 从持久化存储恢复的中断会话
- 从父 Agent 继承的上下文（多 Agent 协作）
- 用于 Few-shot Prompting 的示例消息

为了明确表达上下文的来源和意图，我们引入了 `ContextSource` 类型系统，替代原来不明确的 `InitialMessages` 参数。

### 2.2 ContextSource 类型定义

```csharp
/// <summary>
/// 上下文来源 - 明确表达上下文从哪里来以及为什么
/// </summary>
public abstract record ContextSource
{
    /// <summary>全新对话 - 没有任何历史上下文</summary>
    public sealed record Fresh : ContextSource;

    /// <summary>从持久化存储恢复 - 用于断点续传场景</summary>
    public sealed record Restored(Guid SessionId, ContextSnapshot? Snapshot = null) : ContextSource;

    /// <summary>从父 Agent 继承 - 用于多 Agent 协作场景</summary>
    public sealed record Inherited(
        string ParentAgentId,
        ContextSnapshot ParentSnapshot,
        ContextInheritanceMode InheritanceMode = ContextInheritanceMode.Summary) : ContextSource;

    /// <summary>带历史消息 - 用于 Few-shot Prompting 或测试场景</summary>
    public sealed record WithHistory(
        IReadOnlyList<ChatMessage> Messages,
        string Reason) : ContextSource;

    /// <summary>从快照直接恢复 - 当快照已经在内存中时使用</summary>
    public sealed record FromSnapshot(
        ContextSnapshot Snapshot,
        string Reason = "Direct restore from snapshot") : ContextSource;
}

/// <summary>
/// 上下文继承模式
/// </summary>
public enum ContextInheritanceMode
{
    None,           // 不继承任何上下文
    Summary,        // 仅继承总结（推荐，节省 Token）
    RecentMessages, // 继承最近 N 条消息
    Full            // 继承全部上下文（谨慎使用）
}
```

### 2.3 使用示例

```csharp
// 场景 1：全新对话
var context = new AgentExecutionContext
{
    Input = "分析数据库告警",
    Source = ContextSource.Fresh.Instance  // 默认值
};

// 场景 2：恢复中断的会话
var snapshot = await repository.GetSnapshotAsync(sessionId);
var context = new AgentExecutionContext
{
    SessionId = sessionId,
    Input = "继续之前的分析",
    Source = new ContextSource.Restored(sessionId, snapshot)
};

// 场景 3：子 Agent 继承父上下文（多 Agent 协作）
var parentSnapshot = parentContextManager.ExportSnapshot(parentSessionId);
var context = new AgentExecutionContext
{
    Input = "深入分析网络层问题",
    Source = new ContextSource.Inherited(
        "sre-coordinator",
        parentSnapshot,
        ContextInheritanceMode.Summary)
};

// 场景 4：Few-shot Prompting
var context = new AgentExecutionContext
{
    Input = "分析这个新告警...",
    Source = new ContextSource.WithHistory(
        exampleMessages,
        "输出格式示例")
};
```

```csharp
}
```

```csharp
    var context = new AgentExecutionContext
    {
        SessionId = sessionId,
        Input = "继续之前的任务",
        Source = new ContextSource.Restored(sessionId, snapshot)
    };
    
    return await _agent.ExecuteAsync(context);
}
```

### 3.1 ModelProviderOptions 扩展

```csharp
public class ModelProviderOptions

    /// 各模型的 Token 限制配置
    /// Key: 模型名称, Value: Token 限制信息
    /// </summary>

        }

        return ModelTokenLimits.Default;
    }
    
    /// <summary>获取指定能力级别对应模型的 Token 限制</summary>
    public ModelTokenLimits GetTokenLimits(ModelCapability capability)
    {
        var modelName = GetModel(capability);
        return GetTokenLimits(modelName);
    }
    
    // ... 其他现有方法
}

/// <summary>
/// 模型 Token 限制配置
/// </summary>
public class ModelTokenLimits
{
    /// <summary>模型最大上下文窗口（输入 + 输出）</summary>
    public int MaxContextTokens { get; init; } = 128000;
    
    /// <summary>模型最大输出 Token 数</summary>
    public int MaxOutputTokens { get; init; } = 8192;
    
    /// <summary>
    /// 预留 Buffer 比例（0-1）
    /// 用于预留一定空间给输出和估算误差
    /// </summary>
    public double ReservedBufferRatio { get; init; } = 0.1;
    
    /// <summary>
    /// 计算可用于输入的最大 Token 数
    /// = MaxContextTokens - MaxOutputTokens - Buffer
    /// </summary>
    public int EffectiveInputTokens
    {
        get
        {
            var buffer = (int)(MaxContextTokens * ReservedBufferRatio);
            return MaxContextTokens - MaxOutputTokens - buffer;
        }
    }
    
    /// <summary>默认配置（适用于大多数现代模型）</summary>
    public static ModelTokenLimits Default => new()
    {
        MaxContextTokens = 128000,
        MaxOutputTokens = 8192,
        ReservedBufferRatio = 0.1
    };
    
    /// <summary>Claude 3.5 Sonnet 配置</summary>
    public static ModelTokenLimits Claude35Sonnet => new()
    {
        MaxContextTokens = 200000,
        MaxOutputTokens = 8192,
        ReservedBufferRatio = 0.1
    };
    
    /// <summary>GPT-4o 配置</summary>
    public static ModelTokenLimits Gpt4o => new()
    {
        MaxContextTokens = 128000,
        MaxOutputTokens = 16384,
        ReservedBufferRatio = 0.1
    };
}
```

## 5. Agent 循环中的 Token 检查机制

### 5.1 设计原则

在 Agent 执行 Tool Loop 时，每次发送请求给 LLM 之前，都需要：

1. **估算当前上下文 Token 数**：包括 System Prompt、历史消息、工具结果等
2. **检查是否超出模型限制**：对比模型的 `EffectiveInputTokens`
3. **预留 Buffer 空间**：为输出和估算误差预留空间
4. **触发剪枝**：如果超出限制，调用 `IContextTrimmer` 进行剪枝

### 5.2 Token 检查流程

```
┌─────────────────────────────────────────────────────────────┐
│                    Agent Tool Loop                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 1. 准备发送请求给 LLM                                │   │
│  └─────────────────────────────────────────────────────┘   │
│                          │                                  │
│                          ▼                                  │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 2. 估算当前上下文 Token 数                           │   │
│  │    - System Prompt                                   │   │
│  │    - 历史消息                                        │   │
│  │    - 工具定义                                        │   │
│  └─────────────────────────────────────────────────────┘   │
│                          │                                  │
│                          ▼                                  │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 3. 获取模型 Token 限制                               │   │
│  │    effectiveLimit = MaxContext - MaxOutput - Buffer  │   │
│  └─────────────────────────────────────────────────────┘   │
│                          │                                  │
│                          ▼                                  │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 4. 检查是否需要剪枝                                  │   │
│  │    if (estimatedTokens > effectiveLimit)             │   │
│  │        → 调用 IContextTrimmer.TrimAsync()            │   │
│  └─────────────────────────────────────────────────────┘   │
│                          │                                  │
│                          ▼                                  │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 5. 发送请求给 LLM                                    │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 5.3 ToolLoopAgent 中的实现

```csharp
/// <summary>
/// 支持 Tool Loop 的 Agent 基类
/// </summary>
public abstract class ToolLoopAgent : IAgent
{
    private readonly IContextManager _contextManager;
    private readonly IContextTrimmer _contextTrimmer;
    private readonly ITokenEstimator _tokenEstimator;
    private readonly ModelProviderOptions _modelProviderOptions;
    
    /// <summary>
    /// 执行单次 LLM 调用前的 Token 检查和剪枝
    /// </summary>
    protected async Task<IReadOnlyList<Message>> PrepareMessagesForCompletionAsync(
        ModelCapability capability,
        CancellationToken cancellationToken = default)
    {
        // 1. 获取模型 Token 限制
        var tokenLimits = _modelProviderOptions.GetTokenLimits(capability);
        var effectiveLimit = tokenLimits.EffectiveInputTokens;
        
        // 2. 估算当前上下文 Token 数
        var currentTokens = _contextManager.EstimatedTokenCount;
        
        // 3. 估算工具定义的 Token 数（如果有）
        var toolDefinitionTokens = EstimateToolDefinitionTokens();
        var totalEstimatedTokens = currentTokens + toolDefinitionTokens;
        
        // 4. 检查是否需要剪枝
        if (totalEstimatedTokens > effectiveLimit)
        {
            Logger.LogInformation(
                "Token limit exceeded. Current: {Current}, Limit: {Limit}. Triggering context trimming.",
                totalEstimatedTokens, effectiveLimit);
            
            // 计算剪枝目标（保留一定余量）
            var targetTokens = (int)(effectiveLimit * 0.8) - toolDefinitionTokens;
            
            // 调用剪枝器
            await _contextTrimmer.TrimAsync(
                _contextManager,
                targetTokens,
                cancellationToken);
        }
        
        // 5. 获取剪枝后的消息列表
        return await _contextManager.GetMessagesForCompletionAsync(
            effectiveLimit - toolDefinitionTokens,
            cancellationToken);
    }
    
    /// <summary>
    /// Tool Loop 主循环
    /// </summary>
    protected async Task<AgentResult> ExecuteToolLoopAsync(
        string userInput,
        CancellationToken cancellationToken = default)
    {
        // 添加用户输入
        await _contextManager.AddMessageAsync(new Message
        {
            Role = MessageRole.User,
            Parts = new List<PartBase> { new TextPart { Text = userInput } }
        }, cancellationToken);
        
        while (!cancellationToken.IsCancellationRequested)
        {
            // 准备消息（包含 Token 检查和剪枝）
            var messages = await PrepareMessagesForCompletionAsync(
                ModelCapability.Large,
                cancellationToken);
            
            // 调用 LLM
            var response = await CallLlmAsync(messages, cancellationToken);
            
            // 添加助手响应到上下文
            await _contextManager.AddMessageAsync(response.Message, cancellationToken);
            
            // 检查是否有工具调用
            var toolCalls = ExtractToolCalls(response.Message);
            if (!toolCalls.Any())
            {
                // 没有工具调用，返回最终结果
                return AgentResult.Success(response.Message);
            }
            
            // 执行工具调用
            var toolResults = await ExecuteToolCallsAsync(toolCalls, cancellationToken);
            
            // 添加工具结果到上下文
            await _contextManager.AddMessageAsync(new Message
            {
                Role = MessageRole.Tool,
                Parts = toolResults.Select(r => (PartBase)r).ToList()
            }, cancellationToken);
        }
        
        return AgentResult.Cancelled();
    }
    
    // ... 其他方法
}
```

### 6.2 Framework 提供的剪枝器实现

Framework 提供多个独立的剪枝器实现，每个实现一种策略：

#### 6.2.1 RemoveOldestContextTrimmer（默认）

移除最早的消息，保留最新的消息：

```csharp
/// <summary>
/// 移除最早消息的剪枝器（默认策略）
/// 从后向前保留消息，优先保留最新的对话内容
/// </summary>
public class RemoveOldestContextTrimmer : IContextTrimmer
{
    private readonly ILogger<RemoveOldestContextTrimmer> _logger;
    
    public string Name => "RemoveOldest";
    
    public RemoveOldestContextTrimmer(ILogger<RemoveOldestContextTrimmer>? logger = null)
    {
        _logger = logger ?? NullLogger<RemoveOldestContextTrimmer>.Instance;
    }
    
    public async Task<TrimResult> TrimAsync(
        IContextManager contextManager,
        int targetTokens,
        CancellationToken cancellationToken = default)
    {
        var allMessages = await contextManager.GetAllMessagesAsync(cancellationToken);
        var tokensBefore = allMessages.Sum(m => m.Metadata.EstimatedTokens);
        
        if (tokensBefore <= targetTokens)
        {
            return TrimResult.Success(tokensBefore, tokensBefore, 0, Name);
        }
        
        _logger.LogInformation(
            "Starting {Strategy} trimming. Current: {Current}, Target: {Target}",
            Name, tokensBefore, targetTokens);
        
        var messagesToKeep = new List<Message>();
        var currentTokens = 0;
        var messagesRemoved = 0;
        
        // 始终保留 System 消息
        var systemMsg = allMessages.FirstOrDefault(m => m.Role == MessageRole.System);
        if (systemMsg != null)
        {
            messagesToKeep.Add(systemMsg);
            currentTokens += systemMsg.Metadata.EstimatedTokens;
        }
        
        // 从后向前遍历，保留最新的消息
        var nonSystemMessages = allMessages
            .Where(m => m.Role != MessageRole.System)
            .Reverse()
            .ToList();
        
        foreach (var msg in nonSystemMessages)
        {
            if (currentTokens + msg.Metadata.EstimatedTokens <= targetTokens)
            {
                messagesToKeep.Insert(systemMsg != null ? 1 : 0, msg);
                currentTokens += msg.Metadata.EstimatedTokens;
            }
            else if (!msg.Metadata.IsDeletable)
            {
                // 不可删除的消息强制保留
                messagesToKeep.Insert(systemMsg != null ? 1 : 0, msg);
                currentTokens += msg.Metadata.EstimatedTokens;
            }
            else
            {
                messagesRemoved++;
            }
        }
        
        // 重建上下文
        await contextManager.ClearAsync(cancellationToken);
        await contextManager.AddMessagesAsync(messagesToKeep, cancellationToken);
        
        _logger.LogInformation(
            "{Strategy} trimming completed. Before: {Before}, After: {After}, Removed: {Removed}",
            Name, tokensBefore, currentTokens, messagesRemoved);
        
        return TrimResult.Success(tokensBefore, currentTokens, messagesRemoved, Name);
    }
}
```

#### 6.2.2 PriorityBasedContextTrimmer

按消息优先级剪枝，优先保留高优先级消息：

```csharp
/// <summary>
/// 基于优先级的剪枝器
/// 优先保留高优先级消息，同优先级按时间倒序
/// </summary>
public class PriorityBasedContextTrimmer : IContextTrimmer
{
    private readonly ILogger<PriorityBasedContextTrimmer> _logger;
    
    public string Name => "PriorityBased";
    
    public PriorityBasedContextTrimmer(ILogger<PriorityBasedContextTrimmer>? logger = null)
    {
        _logger = logger ?? NullLogger<PriorityBasedContextTrimmer>.Instance;
    }
    
    public async Task<TrimResult> TrimAsync(
        IContextManager contextManager,
        int targetTokens,
        CancellationToken cancellationToken = default)
    {
        var allMessages = await contextManager.GetAllMessagesAsync(cancellationToken);
        var tokensBefore = allMessages.Sum(m => m.Metadata.EstimatedTokens);
        
        if (tokensBefore <= targetTokens)
        {
            return TrimResult.Success(tokensBefore, tokensBefore, 0, Name);
        }
        
        _logger.LogInformation(
            "Starting {Strategy} trimming. Current: {Current}, Target: {Target}",
            Name, tokensBefore, targetTokens);
        
        // 按优先级排序（高优先级在前），同优先级按时间倒序
        var sorted = allMessages
            .OrderByDescending(m => m.Metadata.Priority)
            .ThenByDescending(m => m.CreatedAt)
            .ToList();
        
        var messagesToKeep = new List<Message>();
        var currentTokens = 0;
        
        foreach (var msg in sorted)
        {
            if (currentTokens + msg.Metadata.EstimatedTokens <= targetTokens || !msg.Metadata.IsDeletable)
            {
                messagesToKeep.Add(msg);
                currentTokens += msg.Metadata.EstimatedTokens;
            }
        }
        
        // 按时间重新排序，保持对话顺序
        messagesToKeep = messagesToKeep.OrderBy(m => m.CreatedAt).ToList();
        
        var messagesRemoved = allMessages.Count - messagesToKeep.Count;
        
        // 重建上下文
        await contextManager.ClearAsync(cancellationToken);
        await contextManager.AddMessagesAsync(messagesToKeep, cancellationToken);
        
        _logger.LogInformation(
            "{Strategy} trimming completed. Before: {Before}, After: {After}, Removed: {Removed}",
            Name, tokensBefore, currentTokens, messagesRemoved);
        
        return TrimResult.Success(tokensBefore, currentTokens, messagesRemoved, Name);
    }
}
```

### 6.3 ToolLoopAgent 中的剪枝器配置

`ToolLoopAgent` 通过 Options 接收 `IContextTrimmer` 实例，并提供默认实现：

```csharp
/// <summary>
/// ToolLoopAgent 配置选项
/// </summary>
public class ToolLoopAgentOptions
{
    /// <summary>
    /// 上下文剪枝器
    /// 如果不指定，使用 RemoveOldestContextTrimmer 作为默认实现
    /// </summary>
    public IContextTrimmer? ContextTrimmer { get; set; }
    
    /// <summary>最大循环次数</summary>
    public int MaxIterations { get; set; } = 50;
    
    /// <summary>剪枝后的目标 Token 比例（相对于模型限制）</summary>
    public double TrimTargetRatio { get; set; } = 0.8;
}

/// <summary>
/// 支持 Tool Loop 的 Agent 基类
/// </summary>
public abstract class ToolLoopAgent : IAgent
{
    private readonly IContextManager _contextManager;
    private readonly IContextTrimmer _contextTrimmer;
    private readonly ITokenEstimator _tokenEstimator;
    private readonly ModelProviderOptions _modelProviderOptions;
    private readonly ToolLoopAgentOptions _options;
    private readonly ILogger _logger;
    
    protected ToolLoopAgent(
        IContextManager contextManager,
        ITokenEstimator tokenEstimator,
        ModelProviderOptions modelProviderOptions,
        ToolLoopAgentOptions? options = null,
        ILogger? logger = null)
    {
        _contextManager = contextManager;
        _tokenEstimator = tokenEstimator;
        _modelProviderOptions = modelProviderOptions;
        _options = options ?? new ToolLoopAgentOptions();
        _logger = logger ?? NullLogger.Instance;
        
        // 使用传入的剪枝器，或创建默认的 RemoveOldestContextTrimmer
        _contextTrimmer = _options.ContextTrimmer 
            ?? new RemoveOldestContextTrimmer();
    }
    
    /// <summary>
    /// 执行单次 LLM 调用前的 Token 检查和剪枝
    /// </summary>
    protected async Task<IReadOnlyList<Message>> PrepareMessagesForCompletionAsync(
        ModelCapability capability,
        CancellationToken cancellationToken = default)
    {
        // 1. 获取模型 Token 限制
        var tokenLimits = _modelProviderOptions.GetTokenLimits(capability);
        var effectiveLimit = tokenLimits.EffectiveInputTokens;
        
        // 2. 估算当前上下文 Token 数
        var currentTokens = _contextManager.EstimatedTokenCount;
        
        // 3. 估算工具定义的 Token 数（如果有）
        var toolDefinitionTokens = EstimateToolDefinitionTokens();
        var totalEstimatedTokens = currentTokens + toolDefinitionTokens;
        
        // 4. 检查是否需要剪枝
        if (totalEstimatedTokens > effectiveLimit)
        {
            _logger.LogInformation(
                "Token limit exceeded. Current: {Current}, Limit: {Limit}. " +
                "Triggering context trimming with {Trimmer}.",
                totalEstimatedTokens, effectiveLimit, _contextTrimmer.Name);
            
            // 计算剪枝目标（保留一定余量）
            var targetTokens = (int)(effectiveLimit * _options.TrimTargetRatio) - toolDefinitionTokens;
            
            // 调用剪枝器
            var trimResult = await _contextTrimmer.TrimAsync(
                _contextManager,
                targetTokens,
                cancellationToken);
            
            if (!trimResult.IsSuccess)
            {
                _logger.LogWarning(
                    "Context trimming failed: {Error}",
                    trimResult.ErrorMessage);
            }
        }
        
        // 5. 获取剪枝后的消息列表
        return await _contextManager.GetMessagesForCompletionAsync(
            effectiveLimit - toolDefinitionTokens,
            cancellationToken);
    }
    
    /// <summary>
    /// Tool Loop 主循环
    /// </summary>
    protected async Task<AgentResult> ExecuteToolLoopAsync(
        string userInput,
        CancellationToken cancellationToken = default)
    {
        // 添加用户输入
        await _contextManager.AddMessageAsync(new Message
        {
            Role = MessageRole.User,
            Parts = new List<PartBase> { new TextPart { Text = userInput } }
        }, cancellationToken);
        
        var iteration = 0;
        while (!cancellationToken.IsCancellationRequested && iteration < _options.MaxIterations)
        {
            iteration++;
            
            // 准备消息（包含 Token 检查和剪枝）
            var messages = await PrepareMessagesForCompletionAsync(
                ModelCapability.Large,
                cancellationToken);
            
            // 调用 LLM
            var response = await CallLlmAsync(messages, cancellationToken);
            
            // 添加助手响应到上下文
            await _contextManager.AddMessageAsync(response.Message, cancellationToken);
            
            // 检查是否有工具调用
            var toolCalls = ExtractToolCalls(response.Message);
            if (!toolCalls.Any())
            {
                // 没有工具调用，返回最终结果
                return AgentResult.Success(response.Message);
            }
            
            // 执行工具调用
            var toolResults = await ExecuteToolCallsAsync(toolCalls, cancellationToken);
            
            // 添加工具结果到上下文
            await _contextManager.AddMessageAsync(new Message
            {
                Role = MessageRole.Tool,
                Parts = toolResults.Select(r => (PartBase)r).ToList()
            }, cancellationToken);
        }
        
        if (iteration >= _options.MaxIterations)
        {
            return AgentResult.Failure($"Exceeded maximum iterations ({_options.MaxIterations})");
        }
        
        return AgentResult.Cancelled();
    }
    
    // ... 其他抽象方法
    protected abstract Task<LlmResponse> CallLlmAsync(
        IReadOnlyList<Message> messages,
        CancellationToken cancellationToken);
    
    protected abstract IEnumerable<ToolCallPart> ExtractToolCalls(Message message);
    
    protected abstract Task<IEnumerable<ToolResultPart>> ExecuteToolCallsAsync(
        IEnumerable<ToolCallPart> toolCalls,
        CancellationToken cancellationToken);
    
    protected abstract int EstimateToolDefinitionTokens();
}
```

### 6.4 业务层自定义剪枝器示例

业务层可以实现自己的 `IContextTrimmer`，例如带 LLM 总结功能的剪枝器：

```csharp
/// <summary>
/// 带 LLM 总结功能的剪枝器
/// 业务层自定义实现
/// </summary>
public class SummarizingContextTrimmer : IContextTrimmer
{
    private readonly IContextTrimmer _fallbackTrimmer;
    private readonly IChatClient _chatClient;
    private readonly ITokenEstimator _tokenEstimator;
    private readonly ILogger<SummarizingContextTrimmer> _logger;
    
    public string Name => "Summarizing";
    
    public SummarizingContextTrimmer(
        IChatClient chatClient,
        ITokenEstimator tokenEstimator,
        IContextTrimmer? fallbackTrimmer = null,
        ILogger<SummarizingContextTrimmer>? logger = null)
    {
        _chatClient = chatClient;
        _tokenEstimator = tokenEstimator;
        _fallbackTrimmer = fallbackTrimmer ?? new RemoveOldestContextTrimmer();
        _logger = logger ?? NullLogger<SummarizingContextTrimmer>.Instance;
    }
    
    public async Task<TrimResult> TrimAsync(
        IContextManager contextManager,
        int targetTokens,
        CancellationToken cancellationToken = default)
    {
        var allMessages = await contextManager.GetAllMessagesAsync(cancellationToken);
        var tokensBefore = allMessages.Sum(m => m.Metadata.EstimatedTokens);
        
        if (tokensBefore <= targetTokens)
        {
            return TrimResult.Success(tokensBefore, tokensBefore, 0, Name);
        }
        
        try
        {
            // 计算需要总结的消息和需要保留的消息
            var (messagesToSummarize, messagesToKeep) = SplitMessages(allMessages, targetTokens);
            
            if (!messagesToSummarize.Any())
            {
                // 没有可总结的消息，退化到默认策略
                return await _fallbackTrimmer.TrimAsync(contextManager, targetTokens, cancellationToken);
            }
            
            // 生成总结
            var summary = await GenerateSummaryAsync(messagesToSummarize, cancellationToken);
            
            // 创建总结消息
            var summaryMessage = new Message
            {
                Role = MessageRole.System,
                Parts = new List<PartBase>
                {
                    new TextPart { Text = $"[历史对话总结]\n{summary}" }
                },
                Metadata = new MessageMetadata
                {
                    Priority = MessagePriority.Important,
                    IsDeletable = false,
                    Tags = new HashSet<string> { "summary" },
                    EstimatedTokens = _tokenEstimator.EstimateTokens(summary)
                }
            };
            
            // 重建上下文
            await contextManager.ClearAsync(cancellationToken);
            
            // 保留 System 消息
            var systemMsg = allMessages.FirstOrDefault(m => m.Role == MessageRole.System);
            if (systemMsg != null)
            {
                await contextManager.AddMessageAsync(systemMsg, cancellationToken);
            }
            
            // 添加总结消息
            await contextManager.AddMessageAsync(summaryMessage, cancellationToken);
            
            // 添加保留的消息
            await contextManager.AddMessagesAsync(messagesToKeep, cancellationToken);
            
            var tokensAfter = contextManager.EstimatedTokenCount;
            
            return TrimResult.Success(tokensBefore, tokensAfter, messagesToSummarize.Count, Name);
        }
        catch (Exception ex)
        {
            _logger.LogWarning(ex, "Summarization failed, falling back to {Fallback}", _fallbackTrimmer.Name);
            return await _fallbackTrimmer.TrimAsync(contextManager, targetTokens, cancellationToken);
        }
    }
    
    private (List<Message> ToSummarize, List<Message> ToKeep) SplitMessages(
        IReadOnlyList<Message> messages,
        int targetTokens)
    {
        var toKeep = new List<Message>();
        var currentTokens = 0;
        
        // 预留 20% 给总结
        var reservedForSummary = (int)(targetTokens * 0.2);
        var availableForMessages = targetTokens - reservedForSummary;
        
        // 从后向前，保留最新的消息
        var nonSystemMessages = messages
            .Where(m => m.Role != MessageRole.System)
            .Reverse()
            .ToList();
        
        foreach (var msg in nonSystemMessages)
        {
            if (currentTokens + msg.Metadata.EstimatedTokens <= availableForMessages)
            {
                toKeep.Insert(0, msg);
                currentTokens += msg.Metadata.EstimatedTokens;
            }
        }
        
        // 剩余的消息需要总结
        var toSummarize = messages
            .Where(m => m.Role != MessageRole.System && !toKeep.Contains(m))
            .ToList();
        
        return (toSummarize, toKeep);
    }
    
    private async Task<string> GenerateSummaryAsync(
        List<Message> messages,
        CancellationToken cancellationToken)
    {
        var conversationText = FormatMessagesForSummary(messages);
        
        var prompt = $"""
            请总结以下对话的关键信息，保留：
            1. 重要的发现和结论
            2. 做出的关键决策
            3. 待处理的问题
            4. 相关的工具调用结果摘要
            
            对话内容：
            {conversationText}
            
            请用简洁的中文输出总结，控制在 500 字以内。
            """;
        
        var response = await _chatClient.CompleteAsync(prompt, cancellationToken: cancellationToken);
        return response.Message.Text ?? "";
    }
    
    private string FormatMessagesForSummary(List<Message> messages)
    {
        var sb = new StringBuilder();
        foreach (var msg in messages)
        {
            var role = msg.Role switch
            {
                MessageRole.User => "用户",
                MessageRole.Assistant => "助手",
                MessageRole.Tool => "工具结果",
                _ => msg.Role.ToString()
            };
            
            foreach (var part in msg.Parts)
            {
                var content = part switch
                {
                    TextPart text => text.Text,
                    ToolCallPart toolCall => $"[调用工具: {toolCall.Name}]",
                    ToolResultPart toolResult => $"[{toolResult.ToolName} 结果: {Truncate(toolResult.Content, 200)}]",
                    _ => ""
                };
                
                if (!string.IsNullOrEmpty(content))
                {
                    sb.AppendLine($"{role}: {content}");
                }
            }
        }
        return sb.ToString();
    }
    
    private static string Truncate(string text, int maxLength)
    {
        if (text.Length <= maxLength) return text;
        return text[..maxLength] + "...";
    }
}
```

```csharp
/// <summary>
/// 默认上下文管理器实现（充血模型）
/// </summary>
public class DefaultContextManager : IContextManager
{
    private readonly List<Message> _messages = [];
    private readonly ITokenEstimator _tokenEstimator;

    private readonly object _lock = new();

    private DefaultContextManager(
        Guid sessionId,

        ContextManagerOptions? options = null)
    {
        SessionId = sessionId;
        _tokenEstimator = tokenEstimator;

    }
    
    // === 工厂方法 ===

        ContextManagerOptions? options = null)

    public void ConfigureTokenLimit(ModelTokenLimits tokenLimits)
    {
        _tokenLimits = tokenLimits;
    }
    
    public void RecordTokenUsage(int promptTokens, int completionTokens)

        // 更新估算并检查是否需要剪枝
        if (_tokenLimits != null)

        }

    {
        if (_tokenLimits == null) return;

        if (EstimatedTokenCount > effectiveLimit)

            var targetTokens = (int)(effectiveLimit * _options.TrimTargetRatio);
            var result = _options.Trimmer.Trim(this, targetTokens);
            // 剪枝完成后更新估算
            EstimatedTokenCount = GetMessages().Sum(m => m.Metadata.EstimatedTokens);

                Role = MessageRole.System,

                    IsDeletable = false,
                    Priority = MessagePriority.Critical,

                }
            };
        }

            return _systemMessage?.Parts.OfType<TextPart>().FirstOrDefault()?.Text;
        }
    }
    
    public void AddUserMessage(string content)
    {
        AddMessage(new Message

            Role = MessageRole.User,
            Parts = [new TextPart { Text = content }]
        });
    }
    
    public void AddAssistantMessage(string content)
    {

        {
            Role = MessageRole.Assistant,

    }
    
    public void AddToolResultMessage(string toolCallId, string toolName, ToolResult result)

            Parts = [new ToolResultPart

                ToolName = toolName,
                IsSuccess = result.IsSuccess,
                Content = result.Content
            }]
        });

    public void AddMessage(Message message)
    {
        lock (_lock)
        {
            // 计算 Token 数
            message.Metadata.EstimatedTokens = _tokenEstimator.EstimateTokens(message);
            
            // 自动压缩长工具结果
            if (_options.AutoCompressToolResults)
            {
                CompressToolResultsIfNeeded(message);
            }
            
            _messages.Add(message);
            EstimatedTokenCount += message.Metadata.EstimatedTokens;
        }
    }

        foreach (var message in messages)
        {
            AddMessage(message);
        }
    }

    {
        lock (_lock)
        {
            var result = new List<Message>();
            if (_systemMessage != null)
            {
                result.Add(_systemMessage);
            }
            result.AddRange(_messages);
            return result;
        }
    }

    {
        lock (_lock)
        {
            _messages.Clear();
            EstimatedTokenCount = _systemMessage?.Metadata.EstimatedTokens ?? 0;
            // 注意：不清除 System 消息
        }
    }
    
    // === 消息输出 ===
    
    public IReadOnlyList<ChatMessage> GetChatMessages()
        => GetChatMessages(DefaultChatMessageBuilder.Instance);

    {
        lock (_lock)
        {
            TrimIfNeeded();
            return builder.Build(GetMessages());
        }
    }

    {
        lock (_lock)
        {
            return new ContextSnapshot
            {
                SessionId = SessionId,
                Version = 1,
                SystemMessage = GetSystemMessage(),
                Messages = _messages.ToList(),
                CreatedAt = DateTime.UtcNow,
                EstimatedTokenCount = EstimatedTokenCount,
                Metadata = metadata ?? new Dictionary<string, object>()

        lock (_lock)
        {
            if (snapshot.SystemMessage != null)
            {
                SetSystemMessage(snapshot.SystemMessage);
            }
            _messages.Clear();
            _messages.AddRange(snapshot.Messages);
            EstimatedTokenCount = snapshot.EstimatedTokenCount;
        }
    }
    
    public string GenerateSummary(int maxTokens = 500)
    {
        // 简单实现：截取最近的消息内容
        var messages = GetMessages().Where(m => m.Role != MessageRole.System).ToList();
        var sb = new StringBuilder();
        
        foreach (var msg in messages.TakeLast(10))
        {
            var text = msg.Parts.OfType<TextPart>().FirstOrDefault()?.Text;
            if (text != null)
            {
                sb.AppendLine($"[{msg.Role}]: {text.Substring(0, Math.Min(100, text.Length))}...");
            }
        }
        
        return sb.ToString();
    }
    
    // === 私有方法 ===
    
    private void CompressToolResultsIfNeeded(Message message)
    {
        foreach (var part in message.Parts.OfType<ToolResultPart>().ToList())
        {
            var tokens = _tokenEstimator.EstimateTokens(part.Content);
            if (tokens > 1000)  // 默认压缩阈值
            {
                var compressed = CompressContent(part.Content, 1000);
                var index = message.Parts.IndexOf(part);
                message.Parts[index] = part with { Content = compressed };
            }
        }
    }
    
    private string CompressContent(string content, int targetTokens)
    {
        var lines = content.Split('\n');
        var result = new List<string>();
        var currentTokens = 0;
        
        foreach (var line in lines)
        {
            var lineTokens = _tokenEstimator.EstimateTokens(line);
            if (currentTokens + lineTokens > targetTokens)
            {
                result.Add("... [content truncated] ...");
                break;
            }
            result.Add(line);
            currentTokens += lineTokens;
        }
        
        return string.Join('\n', result);
    }
}
```

```csharp
/// <summary>
/// 简单的字符估算器
/// 基于字符数估算 Token（约 4 字符/Token）
/// </summary>
public class SimpleTokenEstimator : ITokenEstimator
{
    private const double CharsPerToken = 4.0;
    
    public int EstimateTokens(string text)
    {
        if (string.IsNullOrEmpty(text)) return 0;
        return (int)Math.Ceiling(text.Length / CharsPerToken);
    }
    
    public int EstimateTokens(Message message)
    {
        var total = 0;
        foreach (var part in message.Parts)
        {
            total += part.Type switch
            {
                MessagePartType.Text => EstimateTokens(((TextPart)part).Text),
                MessagePartType.ToolCall => EstimateTokens(((ToolCallPart)part).Name) 
                    + EstimateTokens(((ToolCallPart)part).Arguments),
                MessagePartType.ToolResult => EstimateTokens(((ToolResultPart)part).Content),
                _ => 0
            };
        }
        return total + 4;  // 消息格式化开销
    }
    
    public int EstimateTokens(IEnumerable<Message> messages)
    {
        return messages.Sum(EstimateTokens);
    }
}
```

```csharp
        {

        // 总结消息
        if (_summaryMessage != null)
```
