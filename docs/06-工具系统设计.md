# 工具系统设计

## 1. 概述

工具（Tool）是 AI Agent 与外部系统交互的桥梁。本设计遵循[技术架构设计](./03-技术架构设计.md)中定义的**全局 Result 模式**：

- **Result 模式**：所有工具返回 `ToolResult`（继承自 `ResultBase`），不抛异常
- **错误透明**：错误信息完整传递给 LLM，支持自动重试
- **统一接口**：Framework 定义标准（`ITool`），业务层实现具体工具

> **注意**：`ToolResult` 是全局 Result 模式的一部分，与 `AgentResult`、`LLMResult` 共享相同的基类和设计原则。详见 [03-技术架构设计 - 第3节：全局 Result 模式](./03-技术架构设计.md#3-全局-result-模式)。

## 2. 核心设计：Result 模式

### 2.1 设计原则

传统异常方式的问题：
```csharp
// ❌ 不推荐：抛异常
public async Task<string> ExecuteAsync(...)
{
    var response = await _httpClient.GetAsync(url);
    if (!response.IsSuccessStatusCode)
    {
        throw new ToolExecutionException($"HTTP {response.StatusCode}");
    }
    return await response.Content.ReadAsStringAsync();
}
```

问题：
1. LLM 无法获得详细错误信息
2. 无法判断错误是否可重试
3. 中断了 Agent 的正常流程

Result 模式的优势：
```csharp
// ✅ 推荐：Result 模式
public async Task<ToolResult> ExecuteAsync(...)
{
    try
    {
        var response = await _httpClient.GetAsync(url);
        if (!response.IsSuccessStatusCode)
        {
            return ToolResult.Failure(
                $"HTTP request failed with status {response.StatusCode}. " +
                $"Please check if the endpoint is correct or try again later.",
                errorCode: $"HTTP_{(int)response.StatusCode}",
                isRetryable: response.StatusCode >= HttpStatusCode.InternalServerError);
        }
        var content = await response.Content.ReadAsStringAsync();
        return ToolResult.Success(content);
    }
    catch (HttpRequestException ex)
    {
        return ToolResult.Failure(
            $"Network error: {ex.Message}. Please check connectivity.",
            errorCode: "NETWORK_ERROR",
            isRetryable: true);
    }
}
```

### 2.2 ToolResult 定义

```csharp
/// <summary>
/// 工具执行结果（Result 模式）
/// 无论成功失败，都返回结果而非抛异常
/// </summary>
public class ToolResult
{
    /// <summary>是否成功</summary>
    public bool IsSuccess { get; init; }
    
    /// <summary>
    /// 结果内容
    /// - 成功时：返回的数据/信息
    /// - 失败时：错误描述（供 LLM 理解）
    /// </summary>
    public string Content { get; init; } = string.Empty;
    
    /// <summary>结构化数据（可选，用于程序处理）</summary>
    public object? Data { get; init; }
    
    /// <summary>错误代码（失败时）</summary>
    public string? ErrorCode { get; init; }
    
    /// <summary>
    /// 是否可重试
    /// - true: LLM 可尝试修正参数后重试
    /// - false: 永久性错误，无需重试
    /// </summary>
    public bool IsRetryable { get; init; }
    
    /// <summary>执行耗时</summary>
    public TimeSpan Duration { get; init; }
    
    /// <summary>元数据</summary>
    public Dictionary<string, object> Metadata { get; init; } = new();
    
    #region 工厂方法
    
    /// <summary>创建成功结果</summary>
    public static ToolResult Success(string content, object? data = null)
        => new()
        {
            IsSuccess = true,
            Content = content,
            Data = data
        };
    
    /// <summary>创建成功结果（带元数据）</summary>
    public static ToolResult Success(
        string content, 
        object? data,
        Dictionary<string, object> metadata)
        => new()
        {
            IsSuccess = true,
            Content = content,
            Data = data,
            Metadata = metadata
        };
    
    /// <summary>创建失败结果</summary>
    public static ToolResult Failure(
        string errorMessage,
        string? errorCode = null,
        bool isRetryable = true)
        => new()
        {
            IsSuccess = false,
            Content = errorMessage,
            ErrorCode = errorCode,
            IsRetryable = isRetryable
        };
    
    /// <summary>从异常创建失败结果</summary>
    public static ToolResult FromException(Exception ex, bool? isRetryable = null)
    {
        var retryable = isRetryable ?? IsExceptionRetryable(ex);
        return new ToolResult
        {
            IsSuccess = false,
            Content = $"Tool execution failed: {ex.Message}",
            ErrorCode = ex.GetType().Name,
            IsRetryable = retryable,
            Metadata = new Dictionary<string, object>
            {
                ["exception_type"] = ex.GetType().FullName ?? ex.GetType().Name,
                ["stack_trace"] = ex.StackTrace ?? ""
            }
        };
    }
    
    private static bool IsExceptionRetryable(Exception ex)
    {
        return ex switch
        {
            TimeoutException => true,
            HttpRequestException => true,
            OperationCanceledException => false,
            ArgumentException => false,
            _ => true
        };
    }
    
    #endregion
}
```

## 4. 工具基类

### 4.1 ToolBase

```csharp
/// <summary>
/// 工具基类，提供通用功能
/// </summary>
public abstract class ToolBase : ITool
{
    public abstract string Name { get; }
    public abstract string Description { get; }
    public abstract JsonElement ParameterSchema { get; }
    
    public async Task<ToolResult> ExecuteAsync(
        ToolExecutionContext context,
        CancellationToken cancellationToken = default)
    {
        var sw = Stopwatch.StartNew();
        try
        {
            // 参数验证
            var validationResult = ValidateParameters(context.Parameters);
            if (!validationResult.IsValid)
            {
                return ToolResult.Failure(
                    $"Invalid parameters: {validationResult.ErrorMessage}",
                    "INVALID_PARAMETERS",
                    isRetryable: true);
            }
            
            // 执行具体逻辑
            var result = await ExecuteCoreAsync(context, cancellationToken);
            
            sw.Stop();
            return result with { Duration = sw.Elapsed };
        }
        catch (OperationCanceledException)
        {
            return ToolResult.Failure(
                "Operation was cancelled",
                "CANCELLED",
                isRetryable: false);
        }
        catch (Exception ex)
        {
            sw.Stop();
            return ToolResult.FromException(ex) with { Duration = sw.Elapsed };
        }
    }
    
    /// <summary>子类实现的核心逻辑</summary>
    protected abstract Task<ToolResult> ExecuteCoreAsync(
        ToolExecutionContext context,
        CancellationToken cancellationToken);
    
    /// <summary>参数验证（子类可覆盖）</summary>
    protected virtual ValidationResult ValidateParameters(JsonElement parameters)
    {
        return ValidationResult.Valid();
    }
    
    /// <summary>辅助方法：获取必需的字符串参数</summary>
    protected static (bool Success, string? Value, string? Error) GetRequiredString(
        JsonElement parameters, string propertyName)
    {
        if (!parameters.TryGetProperty(propertyName, out var prop))
        {
            return (false, null, $"Missing required parameter: {propertyName}");
        }
        
        var value = prop.GetString();
        if (string.IsNullOrEmpty(value))
        {
            return (false, null, $"Parameter '{propertyName}' cannot be empty");
        }
        
        return (true, value, null);
    }
    
    /// <summary>辅助方法：获取可选参数</summary>
    protected static T? GetOptional<T>(JsonElement parameters, string propertyName, T? defaultValue = default)
    {
        if (!parameters.TryGetProperty(propertyName, out var prop))
        {
            return defaultValue;
        }
        
        return JsonSerializer.Deserialize<T>(prop.GetRawText());
    }
}

public record ValidationResult(bool IsValid, string? ErrorMessage = null)
{
    public static ValidationResult Valid() => new(true);
    public static ValidationResult Invalid(string error) => new(false, error);
}
```

### 5.1 Sumo Logic 查询工具

```csharp
/// <summary>
/// Sumo Logic 日志查询工具
/// </summary>
public class SumoLogicQueryTool : ToolBase
{
    private readonly ISumoLogicClient _client;
    private readonly ILogger<SumoLogicQueryTool> _logger;
    
    public override string Name => "sumo_logic_query";
    
    public override string Description => """
        Query logs from Sumo Logic.
        Use this tool when you need to search application logs, find errors, or analyze log patterns.
        The query uses Sumo Logic Query Language (similar to SQL).
        """;
    
    public override JsonElement ParameterSchema => JsonDocument.Parse("""
        {
            "type": "object",
            "properties": {
                "query": {
                    "type": "string",
                    "description": "Sumo Logic query string. Example: _sourceCategory=app/production | where level=\"error\""
                },
                "from": {
                    "type": "string",
                    "format": "date-time",
                    "description": "Start time in ISO 8601 format"
                },
                "to": {
                    "type": "string",
                    "format": "date-time",
                    "description": "End time in ISO 8601 format"
                },
                "limit": {
                    "type": "integer",
                    "default": 100,
                    "description": "Maximum number of results to return"
                }
            },
            "required": ["query", "from", "to"]
        }
        """).RootElement;
    
    protected override async Task<ToolResult> ExecuteCoreAsync(
        ToolExecutionContext context,
        CancellationToken cancellationToken)
    {
        // 获取参数
        var (queryOk, query, queryErr) = GetRequiredString(context.Parameters, "query");
        if (!queryOk)
        {
            return ToolResult.Failure(queryErr!, "INVALID_PARAM", isRetryable: true);
        }
        
        var (fromOk, fromStr, fromErr) = GetRequiredString(context.Parameters, "from");
        if (!fromOk)
        {
            return ToolResult.Failure(fromErr!, "INVALID_PARAM", isRetryable: true);
        }
        
        var (toOk, toStr, toErr) = GetRequiredString(context.Parameters, "to");
        if (!toOk)
        {
            return ToolResult.Failure(toErr!, "INVALID_PARAM", isRetryable: true);
        }
        
        // 解析时间
        if (!DateTime.TryParse(fromStr, out var from))
        {
            return ToolResult.Failure(
                $"Invalid 'from' date format: {fromStr}. Please use ISO 8601 format.",
                "INVALID_DATE",
                isRetryable: true);
        }
        
        if (!DateTime.TryParse(toStr, out var to))
        {
            return ToolResult.Failure(
                $"Invalid 'to' date format: {toStr}. Please use ISO 8601 format.",
                "INVALID_DATE",
                isRetryable: true);
        }
        
        var limit = GetOptional<int>(context.Parameters, "limit", 100);
        
        // 验证时间范围
        if (from >= to)
        {
            return ToolResult.Failure(
                "'from' must be earlier than 'to'",
                "INVALID_TIME_RANGE",
                isRetryable: true);
        }
        
        if ((to - from).TotalDays > 7)
        {
            return ToolResult.Failure(
                "Time range cannot exceed 7 days. Please narrow down the range.",
                "TIME_RANGE_TOO_LARGE",
                isRetryable: true);
        }
        
        _logger.LogInformation(
            "Executing Sumo Logic query: {Query}, from {From} to {To}",
            query, from, to);
        
        // 执行查询
        try
        {
            var results = await _client.SearchAsync(query!, from, to, limit, cancellationToken);
            
            if (results.Count == 0)
            {
                return ToolResult.Success(
                    "No logs found matching the query criteria. " +
                    "Consider adjusting the time range or query conditions.",
                    new { count = 0, results = Array.Empty<object>() });
            }
            
            // 格式化结果
            var formatted = FormatResults(results, limit);
            
            return ToolResult.Success(
                formatted,
                new { count = results.Count, results },
                new Dictionary<string, object>
                {
                    ["query"] = query!,
                    ["time_range"] = $"{from:O} - {to:O}",
                    ["result_count"] = results.Count
                });
        }
        catch (SumoLogicQueryException ex)
        {
            return ToolResult.Failure(
                $"Query syntax error: {ex.Message}. Please check the query syntax.",
                "QUERY_SYNTAX_ERROR",
                isRetryable: true);
        }
        catch (SumoLogicRateLimitException ex)
        {
            return ToolResult.Failure(
                $"Rate limit exceeded. Please wait {ex.RetryAfter.TotalSeconds} seconds and try again.",
                "RATE_LIMIT",
                isRetryable: true);
        }
    }
    
    private string FormatResults(List<LogEntry> results, int limit)
    {
        var sb = new StringBuilder();
        sb.AppendLine($"Found {results.Count} log entries:");
        sb.AppendLine();
        
        foreach (var entry in results.Take(Math.Min(limit, 50)))
        {
            sb.AppendLine($"[{entry.Timestamp:yyyy-MM-dd HH:mm:ss}] [{entry.Level}] {entry.Message}");
            if (!string.IsNullOrEmpty(entry.StackTrace))
            {
                sb.AppendLine($"  Stack: {entry.StackTrace.Split('\n').FirstOrDefault()}...");
            }
        }
        
        if (results.Count > 50)
        {
            sb.AppendLine($"... and {results.Count - 50} more entries");
        }
        
        return sb.ToString();
    }
}
```

            }
        }
        
        if (result.Data.Result.Count > 10)
        {
            sb.AppendLine($"... and {result.Data.Result.Count - 10} more series");

        return ToolResult.Success(sb.ToString(), result);

/// <summary>

public class K8sResourceTool : ToolBase
{
    private readonly IKubernetesClient _client;

    public override string Description => """
        Get Kubernetes resource information.
        Use this to check Pod status, Deployment replicas, Service endpoints, etc.
        """;
    
    public override JsonElement ParameterSchema => JsonDocument.Parse("""
        {
            "type": "object",
            "properties": {
                "resource_type": {
                    "type": "string",
                    "enum": ["pod", "deployment", "service", "configmap", "ingress", "event"],
                    "description": "Type of Kubernetes resource"
                },
                "namespace": {
                    "type": "string",
                    "description": "Kubernetes namespace"
                },
                "name": {
                    "type": "string",
                    "description": "Resource name (optional, supports wildcard *)"
                },
                "label_selector": {
                    "type": "string",
                    "description": "Label selector (e.g., app=myapp,env=prod)"
                }
            },
            "required": ["resource_type", "namespace"]
        }
        """).RootElement;
    
    protected override async Task<ToolResult> ExecuteCoreAsync(
        ToolExecutionContext context,
        CancellationToken cancellationToken)
    {
        var (rtOk, resourceType, rtErr) = GetRequiredString(context.Parameters, "resource_type");
        if (!rtOk)
        {
            return ToolResult.Failure(rtErr!, "INVALID_PARAM");
        }
        
        var (nsOk, ns, nsErr) = GetRequiredString(context.Parameters, "namespace");
        if (!nsOk)
        {
            return ToolResult.Failure(nsErr!, "INVALID_PARAM");
        }
        
        var name = GetOptional<string>(context.Parameters, "name", null);
        var labelSelector = GetOptional<string>(context.Parameters, "label_selector", null);
        
        try
        {
            var resources = await _client.GetResourcesAsync(
                resourceType!, ns!, name, labelSelector, cancellationToken);
            
            if (resources.Count == 0)
            {
                return ToolResult.Success(
                    $"No {resourceType} found in namespace '{ns}'" +
                    (name != null ? $" with name '{name}'" : "") +
                    (labelSelector != null ? $" matching labels '{labelSelector}'" : "") +
                    ". Please verify the namespace and resource name.",
                    new { count = 0 });
            }
            
            return ToolResult.Success(
                FormatResources(resourceType!, resources),
                new { count = resources.Count, resources });
        }
        catch (K8sNotFoundException)
        {
            return ToolResult.Failure(
                $"Namespace '{ns}' not found. Please check the namespace name.",
                "NAMESPACE_NOT_FOUND",
                isRetryable: true);
        }
        catch (K8sForbiddenException)
        {
            return ToolResult.Failure(
                $"Access denied to namespace '{ns}'. The agent may not have permission to access this namespace.",
                "FORBIDDEN",
                isRetryable: false);
        }
    }
    
    private string FormatResources(string resourceType, List<K8sResource> resources)
    {
        var sb = new StringBuilder();
        sb.AppendLine($"Found {resources.Count} {resourceType}(s):");
        sb.AppendLine();
        
        foreach (var resource in resources)
        {
            sb.AppendLine($"Name: {resource.Name}");
            sb.AppendLine($"  Status: {resource.Status}");
            sb.AppendLine($"  Created: {resource.CreatedAt:yyyy-MM-dd HH:mm:ss}");
            
            if (resource.Conditions.Any())
            {
                sb.AppendLine($"  Conditions:");
                foreach (var cond in resource.Conditions)
                {
                    sb.AppendLine($"    - {cond.Type}: {cond.Status} ({cond.Reason})");
                }
            }
            sb.AppendLine();
        }
        
        return sb.ToString();
    }
}

### 6.1 工具注册表

```csharp
/// <summary>
/// 工具注册表
/// </summary>
public interface IToolRegistry
{
    void Register(ITool tool);
    ITool? GetTool(string name);
    IEnumerable<ITool> GetAllTools();
    IEnumerable<ITool> GetToolsByTag(string tag);
}

public class ToolRegistry : IToolRegistry
{
    private readonly Dictionary<string, ITool> _tools = new();
    private readonly Dictionary<string, HashSet<string>> _toolTags = new();
    
    public void Register(ITool tool)
    {
        _tools[tool.Name] = tool;
    }
    
    public void Register(ITool tool, params string[] tags)
    {
        _tools[tool.Name] = tool;
        foreach (var tag in tags)
        {
            if (!_toolTags.ContainsKey(tag))
            {
                _toolTags[tag] = new HashSet<string>();
            }
            _toolTags[tag].Add(tool.Name);
        }
    }
    
    public ITool? GetTool(string name) 
        => _tools.TryGetValue(name, out var tool) ? tool : null;
    
    public IEnumerable<ITool> GetAllTools() 
        => _tools.Values;
    
    public IEnumerable<ITool> GetToolsByTag(string tag)
    {
        if (_toolTags.TryGetValue(tag, out var names))
        {
            return names.Select(n => _tools[n]);
        }
        return Enumerable.Empty<ITool>();
    }
}
```

## 7. 错误处理最佳实践

### 7.1 错误分类

| 错误类型 | 可重试 | 示例 |
|----------|--------|------|
| 参数错误 | ✅ | 日期格式错误、缺少必需参数 |
| 语法错误 | ✅ | PromQL/SQL 语法错误 |
| 资源不存在 | ✅ | 命名空间不存在、服务名错误 |
| 超时 | ✅ | 查询超时、网络超时 |
| 限流 | ✅ | API 调用频率限制 |
| 权限不足 | ❌ | 无权访问资源 |
| 服务不可用 | ✅ | 外部服务宕机 |
| 取消 | ❌ | 用户取消操作 |

### 7.2 错误消息规范

```csharp
// ✅ 好的错误消息 - 清晰、可操作
ToolResult.Failure(
    "Invalid date format '2024-13-01'. Please use ISO 8601 format (e.g., '2024-01-15T10:30:00Z').",
    "INVALID_DATE_FORMAT",
    isRetryable: true);

// ❌ 不好的错误消息 - 模糊、无帮助
ToolResult.Failure("Error occurred", "ERROR");
```

### 7.3 重试建议

工具可在错误消息中提供重试建议：

```csharp
// 提供具体的修复建议
ToolResult.Failure(
    $"Query returned too many results ({count}). " +
    "Please narrow down by:\n" +
    "1. Reducing the time range\n" +
    "2. Adding more filters (e.g., specific service name)\n" +
    "3. Using aggregation (e.g., | count by level)",
    "TOO_MANY_RESULTS",
    isRetryable: true);
```
