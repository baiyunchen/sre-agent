### 1.2 核心模块说明

#### 框架层（SreAgent.Framework）

- **ToolLoopAgent**：ReAct 执行引擎，LLM 调用 + 工具执行循环
- **AgentBuilder**：流畅 API 配置 Agent（System Prompt、模型能力、工具、迭代次数等）
- **ToolBase&lt;TParams&gt;**：强类型工具基类，自动生成 JSON Schema
- **DefaultContextManager**：消息存储、Token 限制、裁剪策略
- **ModelProvider**：LLM 客户端管理，支持多供应商
- **SubAgentTool**：将子 Agent 作为工具供父 Agent 调用

#### 应用层（SreAgent.Application）

- **SreCoordinatorAgent**：SRE 故障分析协调器，整合 Todo 与 CloudWatch 工具
- **TodoWriteTool / TodoReadTool**：任务列表 CRUD
- **CloudWatchSimpleQueryTool**：简单日志搜索（时间、日志组、关键字）
- **CloudWatchInsightsQueryTool**：CloudWatch Logs Insights 高级查询

#### API 层（SreAgent.Api）

- **Chat API**：`{ "sessionId"?, "message" }` → 支持新对话与追问
- **Analyze API**：告警结构化输入 → 分析结果 + 任务列表

## 三、建议开发计划（路线图）
